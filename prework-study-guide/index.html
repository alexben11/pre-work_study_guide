HTML Notes

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • p This tag allows you to create paragraphs
  • h This is the largest heading
  • hr This is a horizontal line
  • b This makes text bold
  • i This makes text italic
  • br This tag allows you to insert line breaks
  • a Anchor elements usually used in with reference files
  • ref reference file as in a ref

CSS Notes

  • . is a class selector. ex .card in css, clas="card" in code
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A width indicates with ex. 80% of the available space
  • A margin assigns values to top and bottom first then left and right ex. 40px then auto for left and right
  • A box shadow assigns values on x and y axis as it relates to the element. X is offset of width or extent of shadow. Y is offset on the y axis "below" the element.

Git Notes

  • git checkout -b branch-name: creates a new branch and switches to it
